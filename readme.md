# Code Club RDU — website

Built with [Astro](https://astro.build) and [Tailwind](https://tailwindcss.com).

## Prerequisites

- Node.js `>= 22.12.0`
- pnpm (via `corepack enable` or [pnpm.io](https://pnpm.io/installation))

## Run locally

```sh
git clone git@github.com:codeclubrdu/website.git
cd website
pnpm install
pnpm dev
```

Dev server: <http://localhost:4321>.
